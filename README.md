# CareerFit AI Streamlit Demo

A multi-dimensional career identity and role-fit engine for hackathon demonstration.

## Features

- Big Five personality signals
- MBTI-inspired work-style output
- RIASEC / Holland-style career interests
- Skills profile
- Work values
- Career readiness score
- Role-fit matching engine
- Skill gap diagnosis
- 90-day employability roadmap
- Employer-facing dashboard
- Downloadable candidate report

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then open the local URL shown in the terminal, usually:

```text
http://localhost:8501
```

## Notes

This is a rule-based explainable prototype for hackathon demonstration. It is not a validated psychological assessment. For production use, add psychometric validation, user privacy controls, stronger occupational data, and AI API integration.
